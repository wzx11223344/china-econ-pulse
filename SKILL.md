# china-econ-pulse -- 中国经济脉搏监测系统

## 能力边界

**支持的操作：**

- 自动获取 30+ 项高频中国经济指标（PMI、CPI、PPI、工业增加值、社融、M2、进出口、用电量、货运量等）
- 构建"经济脉搏指数"（Pulse Index）：多指标归一化加权合成综合经济景气指数
- 指标趋势判定（扩张/收缩/中性）与发散检测（多个指标方向矛盾预警）
- 热力图评分：各指标在不同时间窗口上的表现热度
- 基于 Plotly 的专业交互式图表（雷达图、热力图、时间序列对比）
- 双轴对比图表（任意两个指标同框展示，如 PMI vs CPI）
- 生成自包含 HTML 报告（所有 CSS/JS/Plotly 内联，可离线浏览），包含信号仪表盘

**不支持的操作：**

- 实时高频数据流（如分钟级 tick 数据）
- 分省/分城市区域经济指标
- GDP 预测或宏观计量模型
- 股市/债市行情数据
- 指标因果关系检验（Granger 因果等）
- 自定义指标权重配置（使用内置默认权重方案）

## 使用方法

```bash
# 生成完整 HTML 经济脉搏报告
python pulse.py generate

# 指定输出路径
python pulse.py generate --output my_report.html

# 仅显示脉搏指数
python pulse.py pulse

# 列出所有监测指标
python pulse.py indicators

# 在浏览器中显示 PMI 图表
python pulse.py chart pmi

# 双轴对比图表
python pulse.py compare pmi cpi
```

## FAQ

**Q1: Pulse Index 是怎么计算出来的？**
所有指标先做 Z-score 标准化（去量纲），按预设权重加权合成。权重设计参考了传统 PMI 构建方法论与经济逻辑（如 PMI 权重较高，表征"经济温度"；用电量反映实体经济活跃度）。得分 > 50 为扩张，< 50 为收缩。

**Q2: 监测哪些类别的指标？**
覆盖 6 大类：景气调查（PMI 系列）、价格（CPI/PPI）、实体经济（工业增加值/用电量/货运量）、金融（社融/M2/信贷）、外贸（进出口）、消费（社零/汽车销量）。

**Q3: 数据更新频率如何？**
各指标频率不同：PMI 为月频、CPI/PPI 为月频、金融数据为月频，部分高频指标可能为周频。运行命令时自动拉取最新可用数据。

**Q4: HTML 报告包含什么内容？**
报告包含：脉搏指数仪表盘、信号一致性矩阵、热力图、各指标时间序列图、雷达图、发散检测结果。所有图表均为 Plotly 交互式，CSS/JS 完全内联，可在无网络环境下完整浏览。

**Q5: 数据来源是什么？需要 API Key 吗？**
数据通过 akshare 从国家统计局、中国人民银行、海关总署等公开渠道获取，完全免费无需注册。

## 技术栈

- **语言**: Python 3.9+
- **数据源**: akshare（国家统计局、央行、海关等公开数据）
- **科学计算**: NumPy, Pandas
- **可视化**: Plotly（交互式图表，内联到 HTML 报告）
- **模板引擎**: Jinja2（HTML 报告模板渲染）
- **核心模块**: `china_econ_pulse/` (fetcher, indicators, viz, reporter)
- **配置**: `config/indicators.yaml`（指标定义与权重）
- **CLI 入口**: `pulse.py`
